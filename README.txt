Hvordan kører man koden?

Ting du skal bruge for at kunne køre koden: Python, Django, Github og Git. Links er givet under.
Download Django fra den officielle hjemmeside: https://www.djangoproject.com/download/
Derefter følg installationsguiden: https://docs.djangoproject.com/en/4.2/topics/install/
Derudover installer også: https://pypi.org/project/requests/
Git: https://git-scm.com/
Github: https://desktop.github.com/


Når Django er installeret, kan du downloade projektet. For at kører dette projekt skal du igennem et par steps

1.	Ud pak winrar filen eller download github filen - anbefales

Åben cmd vælg via cd stedet du gerne vil have projektet og paste denne kommando: 
git clone https://github.com/simoncclausen/Programmeringseksamen.git

2.	Åben projektet i ”Visual studio code”
3.	Naviger ned til terminalen.
4.	skift directory til der hvor filen er ud pakket ved hjælp af cd
I mit tilfælde er det C:\Users\Bruger\Downloads\Programmering eksamen\mysite (det er vigtigt at du skifter til ”mysite”) så kommandoen lyder ” cd .\mysite\”
5.	Nu skal du skrive ”py manage.py runserver” dette burde starte en localhost

OBS Tjek om du har installeret alle filerne korrekt, og tjek om du har flere versioner af Python.

Hvis der opstår problemer undervejs, referer venligt til: https://docs.djangoproject.com/en/4.2/faq/ eller 
https://www.youtube.com/watch?v=55WQwYntN78
